# Aestheticss

A css template project.

# Prerequisites

1. Node v12.6.0
2. npm@6.9.0
3. pnpm@4.14.0

# Conventions

- Commit Messages
Add these emojis before the commit messages to tag what kind of commit it is.

➕ = additions
✏ = updates
➖ = deleteions

E.g: ➕ Added new CSS utility class
